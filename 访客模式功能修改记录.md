### 1. `src/proxy.ts`

#### 修改 1.1：修改访客模式配置读取路径

**原始代码：**
```typescript
async function getGuestBrowseConfig(request: NextRequest): Promise<boolean> {
  try {
    const url = new URL('/api/server-config', request.url);
    url.searchParams.set('key', 'HomePageConfig');

    const response = await fetch(url.toString(), {
      headers: { 'x-internal-request': 'true' },
    });

    if (response.ok) {
      const data = await response.json();
      if (data.HomePageConfig) {
        return data.HomePageConfig.allowGuestBrowse ?? false;
      }
    }
  } catch {
    // 忽略错误
  }

  return false;
}
```

**修改后代码：**
```typescript
async function getGuestBrowseConfig(request: NextRequest): Promise<boolean> {
  try {
    const url = new URL('/api/server-config', request.url);
    url.searchParams.set('key', 'SiteConfig');

    const response = await fetch(url.toString(), {
      headers: { 'x-internal-request': 'true' },
    });

    if (response.ok) {
      const data = await response.json();
      if (data.SiteConfig) {
        return data.SiteConfig.AllowGuestBrowse ?? false;
      }
    }
  } catch {
    // 忽略错误
  }

  return false;
}
```

**说明：** 将访客模式配置从 `HomePageConfig` 改为从 `SiteConfig` 读取，字段名从 `allowGuestBrowse` 改为 `AllowGuestBrowse`。

**修改后代码：**
```typescript
// 获取认证信息
useEffect(() => {
  if (typeof window !== 'undefined') {
    const auth = getAuthInfoFromBrowserCookie();
    setAuthInfo(auth);

    const guestCookie = document.cookie
      .split('; ')
      .find(row => row.startsWith('guest_mode='));
    setIsGuest(!!guestCookie && guestCookie.split('=')[1] === 'true');
  }
}, []);

// 监听 authInfo 变化，重新检查 guest 状态
useEffect(() => {
  if (typeof window !== 'undefined') {
    const guestCookie = document.cookie
      .split('; ')
      .find(row => row.startsWith('guest_mode='));
    const hasGuestCookie = !!guestCookie && guestCookie.split('=')[1] === 'true';
    const hasAuthInfo = !!authInfo?.username;

    // 如果有认证信息，则不是访客；否则检查 cookie
    setIsGuest(!hasAuthInfo && hasGuestCookie);
  }
}, [authInfo]);
```

**说明：** 添加访客状态检测逻辑，包括初始检测和 authInfo 变化时的重新检测。

## 功能总结

### 新增功能
1. **访客模式开关** - 在后台管理的"站点配置"页面中添加"启用访客模式"开关
2. **访客菜单** - 未登录用户显示简化的用户菜单，只显示"登录"按钮
3. **直播访客支持** - 访客可以访问直播页面和相关 API
4. **直播源 Key 自动填充** - 添加直播源时，输入名称后自动根据拼音首字母生成并填充 Key

### 修改功能
1. **访客配置存储位置** - 从 `HomePageConfig` 移至 `SiteConfig`
2. **登录逻辑** - 登录成功后自动清除访客模式 cookie
3. **用户菜单逻辑** - 根据访客状态动态显示不同菜单
4. **视频源 Key 自动填充** - 已有功能，添加视频源时输入名称自动生成 Key

### 技术细节
1. 使用 `guest_mode` cookie 标识访客状态
2. Middleware 中通过内部请求获取站点配置
3. 登录接口在所有成功路径中清除访客 cookie
4. UserMenu 组件监听 authInfo 变化自动更新访客状态
5. 使用 `pinyin-pro` 库实现中文名称转拼音首字母

## 自动填充 Key 功能

### 功能说明

在后台管理页面添加视频源和直播源时，输入名称后系统会自动根据名称生成 Key 并填充到 Key 输入框中。

**生成规则：**
- 中文字符：取拼音首字母（大写）
- 英文字符：取首字母（大写）
- 数字字符：保留原数字
- 示例："豆瓣" → "DB"，"爱奇艺" → "AQY"，"Netflix" → "N"

### 修改 1：视频源 Key 自动填充（已有功能）

**原始代码：**
```typescript
<input
  type='text'
  placeholder='名称'
  value={newSource.name}
  onChange={(e) =>
    setNewSource((prev) => ({ ...prev, name: e.target.value }))
  }
  className='px-3 py-2 border border-gray-300 dark:border-gray-600 rounded-lg bg-white dark:bg-gray-800 text-gray-900 dark:text-gray-100'
/>
```

**修改后代码：**
```typescript
<input
  type='text'
  placeholder='名称'
  value={newSource.name}
  onChange={(e) => {
    const name = e.target.value;
    const isAdult = /^(AV-|成人|伦理|福利|里番|R18)/i.test(name);
    const autoKey = generateKeyFromName(name);
    setNewSource((prev) => ({
      ...prev,
      name,
      key: autoKey,
      is_adult: isAdult,
    }));
  }}
  className='px-3 py-2 border border-gray-300 dark:border-gray-600 rounded-lg bg-white dark:bg-gray-800 text-gray-900 dark:text-gray-100'
/>
```

**说明：** 输入名称时自动生成 Key 并检测是否为成人内容。

### 核心函数：generateKeyFromName

**代码：**
```typescript
const generateKeyFromName = (name: string): string => {
  if (!name) return '';

  const initials = name
    .split('')
    .map((char) => {
      if (/[a-zA-Z]/.test(char)) return char.toUpperCase();
      if (/[0-9]/.test(char)) return char;
      const result = pinyin(char, { pattern: 'first' });
      return result || char;
    })
    .join('');

  return initials || name.substring(0, 4).toUpperCase();
};
```

**说明：** 
- 使用 `pinyin-pro` 库将中文字符转换为拼音首字母
- 英文字符直接转大写
- 数字字符保持不变
- 如果转换失败，取前 4 个字符转大写作为备用方案
